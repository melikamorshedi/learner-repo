# 🌳 tree Command

## What is it?

`tree` displays directory structure in a hierarchical format.

## Why use it?

- Visualize project structure
- Understand nested directories quickly

## Install

```bash
sudo apt install tree
```

## Usage

### Show full directory tree

```bash
tree
```

### Limit depth

```bash
tree -L 2
```

### Show hidden files

```bash
tree -a
```
