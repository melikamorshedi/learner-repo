## Docker compose 

What is it?

Docker Compose is a tool for defining and running multi-container Docker applications using a YAML file (docker-compose.yml).

#### Why use it?

- Manage multiple containers easily

- Single configuration file

- Simplifies networking between services

  #### Useful Commands

- Start services

```bash
docker compose up -d
```

- #### Stop services

```bash
docker compose down
```


- ####  View log

```bash
docker compose logs -f
```

- #### Example

```bash
docker compose up -d
```
